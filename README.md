# Fitness-Tracker

## Description
 Application using MongoDB and Mongoose that allows a user to track workouts. Uses MongoDB to store the workouts and exercises, and Mongoose to make those calls through Express.js

## Dependencies
 - dotenv
 - express
 - mongoose
 - morgan

## Install
 
 ```
 npm install
 ```

## Seeding

```
npm run seed
```
